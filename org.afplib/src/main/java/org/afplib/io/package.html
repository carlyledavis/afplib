provides classes to read and write AFP.

  • {@link org.afplib.io.AfpInputStream} for reading AFP
  • {@link org.afplib.io.AfpOutputStream} for writing AFP
  • {@link org.afplib.io.AfpFile} as random file access
  • {@link org.afplib.io.AfpMappedFile} reading AFP using NIO mapped files
  • {@link org.afplib.io.AfpFilter} to read, modify and write AFP - all in one go.
